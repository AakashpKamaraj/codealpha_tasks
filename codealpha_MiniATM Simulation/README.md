Mini ATM Simulation System is a Java console-based application that simulates basic ATM operations. The system allows users to securely access their account using a PIN and perform common banking operations such as checking balance, depositing money, and withdrawing money.
This project demonstrates core Java programming concepts including conditional statements, loops, methods, and user input handling. It is developed as part of a CodeAlpha internship task.

Features:

1)Secure PIN login

2)Check account balance

3)Deposit money

4)Withdraw money

5)Automatic balance update after transactions

6)Simple console-based user interface

Technologies Used


1)Java Programming Language

2)Console-based interaction

3)Scanner class for user input


How to Run


Compile the program:

javac MiniATM.java


Run the program:

java MiniATM


Project Type:

Console-based banking simulation application.
